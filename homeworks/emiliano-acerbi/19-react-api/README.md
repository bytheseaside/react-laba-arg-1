## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn add
```

To run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://127.0.0.1:5173/](http://127.0.0.1:5173/) with your browser to see the result.

[Link](https://solvd-calculator.vercel.app/) to the site deployed.
